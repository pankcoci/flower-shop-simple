# 🌸 Райский сад — Упрощённая версия

**Эта версия создана для покупателей. В ней нет админ-панели — только каталог, корзина, заказы и личный кабинет.**

## Функции
## 📦 Что умеет

- Каталог цветов с фото, ценой и скидкой
- Корзина с изменением количества
- Оформление заказа (адрес доставки)
- Личный кабинет: история заказов, статусы
- Редактирование профиля: аватар, имя, телефон, адрес

## Технологии

- Python 3.8+
- Flask
- SQLite
- HTML/CSS

## Установка и запуск

```bash
git clone https://github.com/pankcoci/flower-shop-simple.git
cd flower-shop-simple
pip install flask flask-sqlalchemy flask-bcrypt
python app.py
```
